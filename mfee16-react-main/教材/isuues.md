使用Gist
Gist 是 Github 的一個服務，且任何人都可以使用這個服務，不論你是否擁有 Github 帳號。以開發者的角度來說，最基本的功能就是可以分享片段的原始碼。有別於 Github 的 Repository，Gist 可以只分享專案中的某一個小檔案中的程式碼，且擁有私人的選項，讓別人不會搜尋到你的 Gist。Gist 是免費的服務。
